# @mapleperson/maple-ui

## install

```base
npm install @mapleperson/maple-ui --S
```

### import & use

```js
import Vue from 'vue'
import maple from '@mapleperson/maple-ui'
import "@mapleperson/maple-ui/dist/maple.css"

Vue.use(maple);
```

### template

```html
<maple-icon type="dianzan"/>
```
